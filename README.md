# React Native to Senior Engineer - 12 Week Roadmap

An interactive, self-hosted roadmap for React Native developers to become Senior Mobile Systems Engineers in 12 weeks.

## Overview

This roadmap bridges the gap from React Native Developer to Senior Engineer by adding:

- **Backend Architecture** - PostgreSQL, NestJS, API design
- **AI Integration** - LLMs, streaming, RAG, embeddings
- **Cloud Infrastructure** - AWS, Docker, CI/CD
- **System Design** - Scalable architectures, whiteboard interviews

**Time Investment:** 2 hours/day for 12 weeks  
**Expected Outcome:** $30K-$50K salary increase

## Features

- **Interactive Progress Tracking** - Mark tasks complete, track streaks
- **Pomodoro Timer** - Built-in 25-minute focus sessions
- **Local Storage** - All data saved in browser (no backend needed)
- **Backup/Restore** - Export/import your progress as JSON
- **Notes System** - Add notes to any task
- **Responsive Design** - Works on desktop and mobile

## File Structure

## Quick Start

### Option 1: GitHub Pages (Recommended)

1. Fork this repository
2. Go to Settings → Pages
3. Select "Deploy from Branch" → `main` → `/ (root)`
4. Your roadmap will be live at `https://yourusername.github.io/road-map/`

### Option 2: Local Development

```bash
# Clone the repository
git clone https://github.com/yourusername/road-map.git
cd road-map

# Open in browser (no build step needed)
open index.html

# Or serve with any static server
python3 -m http.server 8000
```

## Data Storage

All progress is stored in browser `localStorage`.

**Keyboard Shortcuts:**

- `Ctrl/Cmd + E` - Export data
- `Ctrl/Cmd + B` - Open backup modal
- `Space` - Start/Pause timer (when timer modal is open)
- `Escape` - Close any modal

## Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6+)
- **No Frameworks** - Pure vanilla for maximum portability
- **No Build Step** - Works directly in browser
